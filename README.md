# Golos - Автоматический кликер бонусов

Скрипт для автоматического нажатия кнопки бонуса на сайте.

## Исправленные ошибки в исходном коде

1. **Константы с заглушками** - заменены на конфигурационный файл
2. **Бесконечный цикл без обработки прерывания** - добавлена обработка Ctrl+C
3. **Отсутствие логирования в файл** - добавлено логирование в `golos.log`
4. **Жестко закодированные селекторы** - вынесены в конфигурацию
5. **Отсутствие проверки успешности операций** - добавлены проверки и возврат статусов
6. **Нет обработки исключений для авторизации** - улучшена обработка ошибок
7. **Потенциальные проблемы с импортами** - создан `requirements.txt`

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте конфигурацию:
   - Скопируйте `config.py` как `config_local.py` (если нужно сохранить пример)
   - Отредактируйте `config.py` с вашими данными

## Конфигурация

Отредактируйте файл `config.py`:

```python
# Настройки сайта
URL = "https://real-site.com"  # Замените на реальный URL

# Учетные данные
LOGIN = "your_real_login"      # Ваш реальный логин
PASSWORD = "your_real_password" # Ваш реальный пароль

# Селекторы (адаптируйте под ваш сайт)
USERNAME_FIELD = "username"          # name атрибут поля логина
PASSWORD_FIELD = "password"          # name атрибут поля пароля
SUBMIT_BUTTON_XPATH = "//button[@type='submit']"  # XPath кнопки отправки
BONUS_BUTTON_CLASS = "bonus-button"  # CSS класс кнопки бонуса
```

## Использование

Запустите скрипт:
```bash
python golos.py
```

Для остановки нажмите `Ctrl+C`.

## Функциональность

- Автоматическая авторизация на сайте
- Обход блокировки кнопок (удаление атрибутов disabled)
- Нажатие кнопки бонуса через JavaScript
- Логирование в файл `golos.log`
- Сохранение скриншотов при ошибках
- Обработка прерывания (Ctrl+C)
- Подсчет успешных попыток

## Структура файлов

- `golos.py` - основной скрипт
- `config.py` - конфигурация
- `requirements.txt` - зависимости
- `golos.log` - файл логов (создается автоматически)
- `error_*.png` - скриншоты ошибок (создаются автоматически)

## Безопасность

- Не коммитьте файл `config.py` с реальными паролями в Git
- Используйте `.gitignore` для исключения конфиденциальных файлов
- Рассмотрите использование переменных окружения для хранения паролей
